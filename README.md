# Hack2Hire - AI Powered Mock Interview Platform

## Overview

Hack2Hire is an AI-powered mock interview platform that helps candidates prepare for technical interviews through personalized interview sessions based on their resume and target job description.

The platform analyzes candidate skills, compares them against job requirements, generates adaptive interview questions, evaluates responses, identifies skill gaps, and provides a detailed interview readiness report.

---

## Features

* Resume Parsing and Skill Extraction
* Job Description Analysis
* Resume-JD Matching Score
* Adaptive Difficulty Adjustment
* AI-Generated Interview Questions
* Real-Time Answer Evaluation
* Skill Gap Detection
* Interview Readiness Scoring
* Detailed Performance Report
* Hosted Web Application

## Project Workflow

1. Upload Resume (PDF)
2. Paste Job Description
3. Analyze Resume and JD
4. Generate Personalized Interview Questions
5. Answer Questions
6. Receive AI Evaluation
7. View Readiness Report and Skill Gap Analysis

## Installation

Clone the repository:

```bash
git clone https://github.com/ayushigupta2132/ai-interview-platform.git
cd ai-interview-platform
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
